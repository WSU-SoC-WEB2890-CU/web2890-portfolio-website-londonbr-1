By Brinlee Rose

This project was part of my User Experience Design class at Weber State Universtiy. We learned about the design process including: understanding (empathize with and define users' problems), explore (ideate and prototype), and materialize (test and implement). As we learned about this process, I applied the principles to the photography company, By Brinlee Rose, to create a website from scratch that would best fulfill user needs.
